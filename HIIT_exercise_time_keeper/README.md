# HIIT Exercise Time Keeper

高強度インターバルトレーニング（HIIT）用のタイマーアプリです。

## 機能

- **エクササイズタイマー**: 運動時間と休憩時間を設定可能
- **縦型プログレスバー**: 現在のエクササイズの進捗を視覚的に表示
- **横型プログレスバー**: 全体セッションの進捗を表示
- **スーパーリロード**: リセットボタン長押しでキャッシュクリア
- **自動バージョン表示**: コミット時に自動更新される軽量バージョン情報

## 開発環境のセットアップ

### Git Hooksの設定

バージョン情報を自動更新するため、以下のコマンドを実行してください：

```bash
./scripts/setup-hooks.sh
```

これにより、コミット時に自動的にバージョン情報が更新されます。

### 手動でバージョン更新

```bash
./scripts/update-version.sh
```

## バージョン表示

アプリの左下に以下の形式でバージョンが表示されます：
- フォーマット: `v2025.07.06-1049-ef2752b`
- 構成: `年.月.日-時分-コミットハッシュ`
